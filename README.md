# Traffic Prediction Dashboard 🚦

This project predicts traffic levels based on time and location using Python and Streamlit.

## Features
- Select location and time
- Predict traffic level (Low, Medium, High)
- Display traffic graph

## Technologies Used
- Python
- Streamlit
- Pandas

## How to Run
1. Install dependencies:
   pip install streamlit pandas

2. Run:
   streamlit run a.py
